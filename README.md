## Progress

 - [x] Unit Testing with Python - Basic Example Using unittest

## Unittest

```bash
python -m unittest -v
```

## Pytest

```bash
python -m pytest --doctest-modules -v
```

## Doctest

Should be in proper folder:
```bash
python -m doctest test_* -v
```
